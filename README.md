# Asem

agent-based modeling framework in Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'asem'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install asem

## Usage

start examples :

```
rackup -p 3001
```
